稳定币时代：Stripe的支付革命

# TLDR

稳定币正迎来规模、监管和传统巨头入场的多重拐点，Stripe 通过收购 Bridge 和 Privy，构建了完整的稳定币支付基础设施。凭借其全球支付网络和新兴市场聚焦，Stripe 在竞争中占据优势，但也面临监管和其他竞争对手的挑战。

# 行业背景：稳定币迎来拐点

稳定币绝对是区块链乃至整个金融科技领域近期的大热点之一，作为web3最具mass adoption效应的产品，它正迎来对于其发展极为关键性的拐点；  
以下将从三个方面去看待稳定币的拐点：

## 1.稳定币的规模与增长

得益于稳定币支付的即时结算，低成本，高度透明等特性，稳定币正在成为全球支付网络的重要补充方案；  
2024年，稳定币链上转账总额为 **$27.6T** ，超过了传统支付巨头 visa 和 mastercard 的总和；  
而2025年，截止本文开始撰写的时间（6.15），稳定币总市值已经突破了 **$250B** ，过去 30 天的转账总额高达 **$4.5T** ，除去 mev 和交易所内交易量后，转账总额为 **$2T** ;

与此同时，在非洲、拉美等新兴经济体，由于高通胀/政治不稳定/信用卡普及率低等原因，稳定币的应用也在迎来显著爆发式增长：非洲市场正以约  **45%** 的年增率领跑全球加密采用，其中稳定币交易已占整个数字资产交易的 **40%** 以上；而在拉美地区，根据 Chainalysis 统计，2023 年 7 月至 2024 年 6 月，区域接受的加密资产总额近 **4,150** 亿美元，其中稳定币交易同比增长约 **42.5%** ，占该年拉美加密总交易额的 **9.1%** ；此外，拉美约 **71%** 的支付服务提供商已部署稳定币用于跨境支付，在非洲，这一比例也在快速提升；

稳定币已从加密资产内部的流动性工具，逐渐演变为具备全球规模效应和真实应用场景的下一代支付基础设施。

## 2\. 监管政策的红利

全球主要经济体正加速为稳定币制定明确的合规框架：

在美国，2025 年提出的 GENIUS Act 正推动联邦层面的稳定币统一监管，明确要求 1:1 储备、定期审计、发行需持牌，并禁止向用户付息。目前已通过参议院关键投票，预计今年内正式立法，届时将成为全球最大的稳定币发行合规市场；

欧盟已于 2024 年底全面实施 MiCA 法案，把稳定币纳入电子货币监管体系，要求储备透明、发行人持有 e-money 牌照，并对欧元区外的跨境发行设定严格门槛。Circle 已在法国获牌，成为首批完全合规的国际稳定币发行方；

香港于2025 年 5 月通过了首部稳定币条例，将于 8 月正式生效，规定稳定币发行必须获牌，符合储备和赎回等要求，蚂蚁国际等已宣布申请。

## 3\. 传统巨头入场趋势

近两年，全球支付巨头正加速将稳定币纳入支付与结算环节；

以 Visa 和 Mastercard 为例，两大卡组织先后在 2023–2025 年期间推进了多项稳定币结算试点。Visa 先后与 Circle（USDC） 和 Solana 合作，将 USDC 用于跨境结算和收单清算，显著提高结算速度并降低中间费用。Mastercard 则与多家加密钱包及交易所合作，推出支持用户用稳定币直接消费的加密卡，覆盖全球超过 **1.5** 亿家商户。

PayPal 自 2023 年起推出自有美元稳定币 PYUSD，并将其接入到自家钱包、商户收单和 P2P 转账等服务场景；2025 年，PYUSD 流通市值已突破 **10** 亿美元，并扩展至 Solana 和 Stellar 等高效链上网络，进一步降低用户转账成本。

零售与电商巨头也在跟进；2025 年一季度，沃尔玛 宣布在美国及墨西哥的部分门店试点 USDC 付款，主要用于无银行卡人群的小额支付；亚马逊 则已在 AWS 及部分海外站点测试稳定币结算，方便跨境卖家快速收款、降低换汇成本；而近期，这两家更是传出新闻，要推出自己家的稳定币；

在此趋势下，Stripe 作为全球领先的支付和金融基础设施公司，通过先后收购Bridge和Privy等措施，加速其稳定币的布局，确保其在全球跨境支付和 Web3 融合场景下继续保持技术和产品的领先地位。

# Stripe的战略演进

“Stablecoins are room-temperature superconductors for financial services.”  
在24年底，Stripe 开启对 Brdige 收购洽谈流程后，Stripe 的创始人兼CEO Patrick Collison在X上发了这句话，背后表现的是这位金融科技企业家对于稳定币的长期坚定信仰；

如果你熟悉 Stripe 的企业文化，你一定不会对它骨子里的「远见卓识」感到意外，因为团队习惯于用更长的周期去思考问题；创始人 Collison 兄弟常说，做决策要拉长视角，看十年、二十年后世界的样子；

在过往的年终总结里，Collison兄弟还曾引用查理·芒格的箴言提醒团队—“找到一个简单的想法，然后极其认真地对待它。”

在 Stripe 看来，这个“简单的想法”就是**让支付成为互联网的原生能力（primitive）**，让资金流像数据包一样被路由、编排、组合，商业创新就不再受地理、货币与管道的限制；

事实上，Stripe 也确实在不断投入这个目

标：

* 2010 年，两兄弟在旧金山公寓写下七行代码，首笔交易随即跑通；如今，Stripe 每天处理 **5 亿+ API 请求**，峰值 13 000 次/秒 。

* 2024 年，平台总支付量达到 **1.4 万亿美元**，相当于全球 GDP 的约 1.3%，同比增长 38% ，公司估值为 650亿美元。

* Stripe 已服务半数 Fortune 100、80% Forbes Cloud 100、78% Forbes AI 50 ；全球有 **138 万** 活跃网站正在调用其接口，累计历史接入站点超 450 万 。

* 在 47 个国家本地收单、135 种货币/支付方式之外，Paystack 等伙伴又把覆盖面延伸到 50 \+ 新兴市场 。

当时代的两大风口：AI与web3来临时，Stripe也从未停下学习发展与运用的步伐；Sessions 2025 上发布的全球首个“支付基础模型”借 tens of billions 交易训练，已将大型客户的卡测试攻击检出率从 59% 提升到 97% ；同一舞台上亮相的，是基于稳定币的 Financial Accounts——企业在 101 国可开立美元计价余额，通过 ACH、SEPA 和 8 条公链收付 USDC/USDB ；

如果说 AI 让“识别一笔好交易”变得更聪明，稳定币则让“把钱真正结算出去”变得近乎即时；

下面这条时间轴，记录了 Stripe 如何逐步融合链上美元——

* 2024-10-10：开放 USDC 支付（Ethereum / Solana / Polygon），美国商家当天即可接入 

* 2024-10-16：彭博披露 Stripe 正洽谈收购稳定币基础设施公司 Bridge，为其链上清算能力铺路 

* 2025-02-04：以 11 亿美元完成 Bridge 收购，为 Stripe 历史最大并购 

* 2025-04-30：Bridge 与 Visa 推出拉美首批稳定币借记卡，链上余额即刷即兑法币，计划迅速扩展至欧非亚 

* 2025-05-07：Sessions 发布 Stablecoin Financial Accounts，面向 101 国开放链上美元账号 

* 2025-06-11：Stripe 宣布收购钱包基础设施商 Privy，其 7 500 万嵌入式钱包账户与 Stripe 账户体系天然对接   
* **2025-06-12**：携手 Shopify，让 34 国数百万商家支持 USDC 结算；Stripe Connect 平台可一键为下游开放稳定币收款 

前有Bridge，后有Privy，这两起并购案，吸引了整个fintech圈的关注；

宏观来看，让 Stripe 拥有了“后端清算—账户中台—钱包前端”三位一体的栈，相当于在稳定币支付赛道做了垂直整合：Bridge 负责链上清算与流动性池，Financial Accounts 把它们封装成企业熟悉的余额体系，而 Privy 则为开发者提供无摩擦的钱包入口；值得一提的是，Bridge 和 Privy 都是以API的形式来提供服务的，这种模式的好处就是简单易用，而且前期成本更低；这也反映出 Stripe 的理念：当链上资产可以像 API 一样被调用，支付网络的摩擦就会像电阻被降到几乎为零；

如果稳定币能比现有系统更好地满足更快的结算、更低的费用、更广的可及性这三点，它就很大概率会成为革命性的新基础设施；

于是乎，Stripe开始逐步将“链上美元”纳入其花十年时间建立起的全球支付网关系统之中。

# Bridge 与 Privy 的价值剖析

Bridge成立于2022，定位是基于API的稳定币支付平台，用户只需用少量代码，集成Bridge所提供的相关API之后，即可享受完整的稳定币支付能力；Bridge在2024年8月已完成5800万美元的融资，其客户包括Coinbase、SpaceX 等知名公司；

从功能来看，Bridge 提供三大核心服务：一是Orchestration API，通过几行代码就能移动、存储和接收稳定币，自动处理合规与清算细节；二是Issuance API，开发者可快速发行自有稳定币，Bridge 将相应储备投入美国国债并与开发者共享收益；三是Money Transfer，帮助企业实现全球资金转移并提供美元、欧元账户 ；

Stripe收购Bridge，价值主要体现在以下两个方面：

- 技术优势： Bridge 的 API 模型允许企业通过少量代码集成稳定币支付，无需管理底层服务。这与 Stripe 的现有支付 API 策略高度一致，类似于 Plaid、Twilio 或 AWS 的集成方式  
- 市场定位： 稳定币交易量激增，Stripe 通过 Bridge 可以快速进入这一增长的市场，特别是在跨境支付领域。稳定币提供显著的速度、覆盖范围和成本优势，使其成为全球多种用例的实用选择

Privy 则致力于解决加密钱包的使用痛点。该公司2021年成立，通过一个统一API让开发者可在应用内创建和管理钱包，用户无需接触私钥或外部扩展，这使得复杂的加密操作被“抽象掉”，让加密资产管理变得直观安全；Privy 的增长同样惊人：目前其平台已支撑超过7,500万条钱包账户，为1000多家开发团队服务，日均交易额达数十亿美元 ，其代表性客户包括Hyperliquid、Toku和Blackbird Labs等加密应用；

Privy 的收购，使得 Stripe 可以直接将其简洁的钱包API嵌入到已有的稳定币账户中，带来的是用户体验上的优化；

总结一下，Bridge 让 Stripe 接入了稳定币支付的轨道，而 Privy 则让 Stripe 的用户更容易使用这一套轨道；自此以后，Stripe 拥有了强大的基础设施，整合了 ToB 和 ToC ，再配合它本就拥有的扎实的传统支付背景，拥有了一条完整的稳定币+法币的支付链路；

可以说，这两起收购所造成的影响绝对不容小觑，我们可以预见，日后还会出现其他的“传统巨头收购web3公司”事件，让传统支付和crypto之间的分界线变得愈发模糊。

综合来看，Stripe 的竞争优势体现在三点：

(1) 商户网络规模——年处理 1.4 万亿美元交易，稳定币功能可第一时间下沉至真实消费场景；

(2) 垂直整合技术栈——Bridge (链上清算) \+ Financial Accounts (余额中台) \+ Privy (嵌入式钱包) 构成端到端稳定币解决方案，开发者仅需几行代码即可完成收付；

(3) 先发合规筹码——多边支付牌照矩阵，为大型机构与跨境企业降低监管不确定性。

面临的关键挑战主要包括：美国及各国稳定币法规尚在落地过程中的变数；公链 Gas 波动和跨链安全事件带来的体验与信任风险；Visa、Mastercard、PayPal (已推出PYUSD)乃至Amazon等巨头正（计划）推出自家闭环稳定币方案，对商户心智和流量分流；Stripe 需继续强化合规与安全投入，并进一步凸显“前中后三位一体的全栈优势，方能稳固其优势地位。
